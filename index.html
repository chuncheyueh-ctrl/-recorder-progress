雲端同步狀態

尚未連線

目前一定會先存在本機 Safari。若未設定 Firebase，就是「本機版」；設定 Firebase 後，才會同步到雲端。

Firebase 設定

請把 Firebase 網頁設定貼在右邊。你只需要設定一次；之後會自動儲存在本機。

設定步驟

1. 到 Firebase 建立專案。
2. 建立 Firestore Database，模式可先選測試模式。
3. 專案設定 → 你的應用程式 → 新增 Web App。
4. 複製 firebaseConfig，貼回左邊。
5. 按「儲存設定並連線」。

提醒：測試模式有期限，正式長期使用時要調整 Firestore Rules。這版不做公開學生登入，只作老師使用的雲端同步。

建議 Firestore Rules（老師自用測試版）

如果只是你自己先測試，可以先用 Firebase 預設測試模式。正式公開前，不建議長期使用開放規則。

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /battlePanflute/{docId} {
      allow read, write: if true;
    }
  }
}
